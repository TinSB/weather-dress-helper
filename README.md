## 智能穿衣助手

这是一个我自己写来「解决每天出门前要纠结穿什么」的小工具。

思路很简单：  
**环境温度 + 衣物保暖值 + 体感修正 ≈ 你今天真实感觉到的冷热。**

我把常见的上装、下装抽象成一个个「保暖分」，再结合实时天气（温度、风、湿度、体感温度），用 26°C 作为一个相对舒服的体感基准，算出你**还差多少“保暖度”**，然后用衣服把这部分补回来。

---

### 这个小 app 能做什么

- 实时获取天气  
  - 支持自动定位，也可以自己选城市（中英文搜索都可以）。  
  - 显示气温、体感温度、湿度、风速和大致天气情况。

- 体感修正  
  - 风大、阴冷、太阳直晒、运动量大，这些都可以一键勾选。  
  - 每个条件对应一个温度修正，用来更贴近真实体感，而不是仅看气温。

- 数字衣橱  
  - 上装 / 下装分开，常见单品都有（短袖、衬衫、卫衣、毛衣、大衣、棉裤等）。  
  - 每件衣服都有一个保暖系数，点击勾选后会叠加到总保暖值里。  
  - 界面会同时显示：
    - 当前环境 + 衣物叠加后的体感偏冷 / 偏热  
    - 还差多少保暖值比较合适

- 简单可视化  
  - 用 Chart.js 把不同衣物的保暖系数画成条形图。  
  - 能直观看出哪几件衣服是「轻薄过渡」，哪几件是「重型装备」。

整体目标只有一个：  
> 让「穿多穿少」这件事，变成一次算得清楚的小决策，而不是反复试错。

---

### 大致技术栈

- UI：Tailwind CSS + 一些自定义动画  
- 图表：Chart.js  
- 天气和地理数据：
  - Open-Meteo（天气 & 城市搜索）
  - ipwho.is + 浏览器 Geolocation（定位）
- 纯前端实现，可以直接挂在 GitHub Pages 或任意静态托管上
